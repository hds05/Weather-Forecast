# weather-Forecast

This project is about Weather Forecast. In Allows user to search about weather forecast by city name ot they can search for their current location.

##Features
- Integration with Weather API
- Location-Based Forecast
- User can search for weather forecasts by city name.
- User can search weather forecasts for their current location.
- Extended weather forecasts for multiple days (e.g., 5-day forecast).
- Location-specific weather forecasts.
- Temperature unit toggle (°C/°F) on today's temprature.
- Responsive on multiple devices.
- Interactive UI.

##Installation

```terminal```

##Usage
  Users can search weather forecast of multiple cities as well as current location.
